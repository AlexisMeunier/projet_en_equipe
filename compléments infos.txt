#### Commit de la structure d'un site web vide :

Le fichier .gitkeep permet de commiter des dossiers "vides" puisque les dossiers n'existent pas réellement sur git (ils sont virtuels).

Pour palier à ce "problème", on peut créer un fichier .gitkeep (vide) afin d'avoir une structure de base
